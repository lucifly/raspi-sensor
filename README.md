# raspi-sensor
# 使用raspberryPi作为物联网网关将传感器IP化
# 同时读取多个传感器数据并向订阅者广播

## 涉及到的传感器模块列表
| 模块                 | 传感器               | 型号     |
| -------------------- | -------------------- | -------- |
| 人体红外感应模块     | 热释电红外传感器     | HC-SR501 |
| 温湿度模块           | 数字温湿度传感器     | DHT11    |
| 常闭型震动传感器模块 | 震动开关             | SW-420   |
| 测距模块             | 超声波传感器电子模块 | HC-SR04  |
